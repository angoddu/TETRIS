# TETRIS
Fall 2014 Project
Created by Andrew Goddu

Controls:
Up Arrow: Rotate shape
Down arrow: drop shape one block
Left/Right arrow: Shift shape left/right
P: Pause/Resume game
R: Play again (only available when the current game has been lost)

Scoring:
One row: 40 points
Two rows: 100 points
Three rows: 300 points
Four rows: 1200 points

Leveling and speed
Level increaces every 10 rows cleared
Shapes fall at 1 block per second (1000 mil).
Every level the speed increaces by 20%
Top speed is 10 blocks per second (100 mil)
